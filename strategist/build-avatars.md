description: Build 3+ advertising avatars per client for DCT targeting. 12-point psychological breakdown per avatar. Requires buyer-language dossier. Use at start of any ad campaign.
---

Build advertising avatars for $ARGUMENTS.

1. Read dossier at clients/<project>/research/buyer-language-dossier.md
2. Read ICP at clients/<project>/icp.md
3. Identify 3+ distinct buyer segments from the data (not assumed — evidence-based)
4. Per avatar, build 12-point breakdown:
   - Name + demographic snapshot
   - Core pain (verbatim from dossier)
   - Desired outcome
   - Trigger event
   - Current solution
   - Objections
   - Fear
   - Awareness level (Schwartz)
   - Trusted voices
   - Where they congregate
   - Emotional anchor
   - Messaging angle

Output: clients/<project>/avatars/avatar-1.md, avatar-2.md, avatar-3.md

Each avatar must be distinct — no overlapping pain points across avatars.
DM Copywriter + self (for ad-concepts) when avatars are done.
