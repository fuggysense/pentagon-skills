Run keyword research for $ARGUMENTS.

1. Read clients/<project>/icp.md + dossier to understand buyer search language
2. Query DataForSEO: search volumes, related keywords, "people also ask", SERP features
3. Cross-reference with dossier — real buyer phrases > assumed keywords
4. Group keywords by:
   - Intent: informational / commercial / transactional
   - Awareness level: unaware / problem-aware / solution-aware / product-aware
   - Priority: high volume + low difficulty + high intent first
5. Output keyword map with columns: keyword, volume, difficulty, intent, awareness level, recommended content type

Output: clients/<project>/seo/keyword-map.md
