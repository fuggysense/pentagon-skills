2. Voice match — compare against voice/jerel/ and clients/<project>/brand-voice.md
3. Buyer language — replace generic phrases with dossier Power Words
4. Forbidden words — flag and replace any from dossier Forbidden Words list
5. De-AI sweep — remove: "delve", "tapestry", "leverage", "robust", "comprehensive", "game-changer", "In today's...", "When it comes to...", corporate hedging
6. Mobile format — break paragraphs >2 sentences
7. Read aloud test — if it sounds like a LinkedIn influencer, rewrite
